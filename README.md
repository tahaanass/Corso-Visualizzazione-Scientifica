# Corso-Visualizzazione-Scientifica
Presentazione di un progetto dato API Spotify

## Descrizione  
Analisi statistica e visualizzazione scientifica di più di 100.000 brani per identificare l'identità acustica delle hit di successo.

---

##  Dataset  
Fare riferimento alla cartella: Dataset  
I dati sono stati scaricati dai seguenti siti:

[Hugging Face](hf://datasets/maharshipandya/spotify-tracks-dataset/dataset.csv)

---

## Librerie Usate  
Per il progetto abbiamo usato per le librerie di **Python**:
  - `pandas` : per la manipolazione e l'analisi dei dati.
  - `numpy` : per il calcolo scientifico e matematico sui dati. 

## Data Visualization
  - `matplotlib` : per creare visualizzazioni statiche di base (come grafici a linee, a barre o istogrammi).
  -  `seaborn` : per creare grafici statistici molto più belli, moderni e complessi con poche righe di codice.
  - `plotly` : per creare grafici interattivi
## Utility del Sistema
  - `datetime` : per gestire, formattare e fare calcoli con le date e gli orari.
